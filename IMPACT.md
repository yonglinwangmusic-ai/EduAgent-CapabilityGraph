# Impact & Social Contribution
## Project Context
This project was delivered as a pro-bono technology consulting engagement for A Better Community (ABC), in partnership with Polus International College (四川国际标榜职业学院). I volunteered approximately [x] hours over 7 months Sep 2025 – Feb 2026 as Solution Architect, outside of my regular employment.

### The Challenge
Polus International College operates 5 schools with 50 programs serving 17,000+ students, supported by ~600 faculty. Aligning curricula with market demand across all 50 programs is a massive undertaking — manual curriculum review takes 2+ months per program, making full-coverage analysis practically impossible. There was no structured, data-driven pipeline to bridge recruitment market intelligence and educational program design.

### My Contribution
I designed the complete system architecture for a domain-agnostic AI Agent pipeline that transforms fragmented recruitment data into structured capability graphs. This was a 0-to-1 cold start — no prior system or methodology existed.
#### Specific architectural contributions:
- 4-stage analytical framework (Framework Construction → Data Preparation → Mapping & Association → Analysis & Application)
Dual unique identifier system (A-code / B-code) for data lineage and traceability
3-step AI semantic reasoning pipeline for role normalization, capability extraction, and skill deduplication
Universal capability graph schema referencing national vocational standards, WorldSkills, and OBE methodology
Phased rollout strategy from single pilot to 50-program institutional coverage
Domain configuration separation enabling reuse across any vocational discipline

I was not involved in code implementation, deployment, or day-to-day data operations.
## Direct Impact
#### Architecture Adoption

Architecture adopted and implemented by the project team using Coze (ByteDance) as the agent orchestration platform
Medical aesthetics pilot completed: 388 JDs processed → 6 standardized roles → 18 capability modules
Interactive knowledge graph, skill radar, and career path tree generated and delivered to partner institution

### Scalability Established

Architecture designed to serve all 50 programs at Polus International College — each new program requires only domain configuration changes, not system redesign
Methodology validated through the pilot as a replicable pattern for vocational education AI applications

### Social Impact

Enabled a nonprofit (ABC) to deliver technology-driven workforce development solutions to vocational education
Contributed to bridging the education-industry gap for underrepresented job seekers and vocational students
Open-sourced the architecture to allow other organizations to replicate this approach
